# File: a_info.txt


# This file is used to initialize the "lib/raw/a_info.raw" file, which is
# used to initialize the "artifact" information for the Angband game.

# Do not modify this file unless you know exactly what you are doing,
# unless you wish to risk possible system crashes and broken savefiles.

# The artifact indexes are defined in "defines.h", and must not be changed.

# Artifacts 1-21 are amulets, rings, and lights, 22-96 are armours, 97-185 
# are weapons, and 186-209 are backpack items.  There are plenty of free
# indexes after each base object kind, so you can add new shields after old.

# Artifacts come with descriptions that appear when they are normally 
# identified.  Special thanks to J.R.R Tolkien, without whom the words would 
# be unwritten, the images unconceived, the deed undone. -LM-
# Contributers: Jeff Butler, Neal Hackler, Ethan Sicotte, Pat Tracy, Yuanli Zhou

# Hack -- "Grond" and "Morgoth" MUST have a rarity of one, or they might
# not be dropped when Morgoth is killed.  Note that they can never be 
# created in any other way.



# Version stamp (required)

V:1.1.6

#N: artifact index: name
#I: base object tval (general kind): base object sval (specific type): 
#     number to modify stats or certain skills by (pval)
#W: depth: rarity: weight in deci-pounds: value
#P: base bonus to armour class: damage done when wielded or thrown 
#     (or the damage multiplier, for missile launchers): 
#     Bonus to Skill: Bonus to Deadliness: Bonus to armour class: 
#     Activation Index (0 if item does not activate)
#F: All item-specific flags (special qualities).  Some use the pval number.
#D: Artifact descriptions.
